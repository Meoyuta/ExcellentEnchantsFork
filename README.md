## 特性

### 集成与兼容性
- [**无缝集成**](https://nightexpressdev.com/excellentenchants/enchantments/integration/)。完全兼容原版机制、命令和其他插件。
- **铁砧与砂轮支持**。可以像原版附魔一样合并或移除自定义附魔。
- **附魔书支持**。直接从创造模式菜单中获取自定义附魔书。
- [**PlaceholderAPI**](https://nightexpressdev.com/excellentenchants/placeholders/papi/) 支持。

### 附魔获取途径
- [**附魔台**](https://nightexpressdev.com/excellentenchants/enchantments/distribution/) **与村民交易**。通过标准游戏进程获得自定义附魔。
- [**随机战利品**](https://nightexpressdev.com/excellentenchants/enchantments/distribution/)。在地牢箱子中找到附有自定义附魔的物品。
- [**钓鱼**](https://nightexpressdev.com/excellentenchants/enchantments/distribution/)。钓上带有自定义附魔的附魔书。
- [**生物装备**](https://nightexpressdev.com/excellentenchants/enchantments/distribution/)。生物可以装备带有自定义附魔的装备生成。

### 自定义与控制
- **高度可定制**。修改任意附魔的属性。
- **超等级附魔**。允许附魔超过最高等级限制。
- **互斥设定**。为每个附魔定义不兼容的附魔。
- [**禁用附魔**](https://nightexpressdev.com/excellentenchants/enchantments/disabling/)。全局或按世界禁用附魔。

### 物品与装备支持
- **斧支持**。将剑类附魔应用于斧。
- **弩支持**。默认将弓类附魔应用于弩。
- **鞘翅支持**。默认将胸甲附魔应用于鞘翅。
- [**物品列表**](https://nightexpressdev.com/excellentenchants/enchantments/item-sets/)。为附魔创建自定义的主物品和支持物品列表。

### 用户界面与展示
- **附魔 GUI**。可自定义的图形界面，用于浏览所有自定义附魔。
- **彩色提示框**。自定义附魔的提示框颜色。
- [**描述提示框**](https://nightexpressdev.com/excellentenchants/features/tooltips/)。在物品提示框中显示附魔简介。
- **视觉效果**。附魔包含粒子效果和声音效果。

### 进阶机制
- [**附魔充能**](https://nightexpressdev.com/excellentenchants/features/charges/)。为附魔引入基于充能次数的机制。
- [**新诅咒**](https://nightexpressdev.com/excellentenchants/enchantments/list/)。在原版之外增加额外的诅咒附魔。

## 需求

你必须安装 [NightCore](https://nightexpressdev.com/nightcore/) 插件才能运行 **ExcellentEnchants**。Minecraft 26.1.2 支持需要 NightCore 2.15.2 或更新版本。

支持以下版本和平台：

| **服务端版本** | **Paper** | **Spigot** | **Folia** | **Java 版本** |
|:------------------:|:---------:|:----------:|:---------:|:----------------:|
|       26.1.2       |    是    |    是     |    否     |        25        |
|      1.21.11       |    是    |    是     |    否     |        21        |
|      1.21.10       |    是    |    是     |    否     |        21        |
|       1.21.9       |    是    |     否     |    否     |        21        |
|       1.21.8       |    是    |    是     |    否     |        21        |

- 兼容性表格中未列出的任何内容均**不**支持。
- 请务必查看所有已知问题和不兼容情况 [在此](https://nightexpressdev.com/excellentenchants/common-issues/)。

**可选插件：**
- [PacketEvents](https://spigotmc.org/resources/80279/) - 用于物品提示框中的附魔描述（只需其一）。
- [ProtocolLib](https://ci.dmulloy2.net/job/ProtocolLib/) - 用于物品提示框中的附魔描述（只需其一）。
